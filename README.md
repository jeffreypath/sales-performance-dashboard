# Sales Performance Dashboard | Power BI

## Overview

A 3-page interactive Power BI report analysing retail sales performance across regions, stores, managers, and salespersons. Built end-to-end from raw transactional data to executive-ready visuals using Power BI and DAX.

## Dashboard Pages

Page 1 - Sales Performance: High-level KPIs including total revenue before and after discount, average ticket size, and item return rate. Includes monthly sales trend by region and revenue breakdown by product type.
Page 2 - Salesperson Performance: Manager and salesperson comparison with discount impact analysis. Filterable by manager and region to drill into individual performance.
Page 3 - Data Appendix: Fully filterable transaction-level table with slicers for region, store, manager, salesperson, and date range. Designed for ad-hoc investigation and data validation.

## Tools Used

Power BI, DAX, Microsoft Excel

## Data Source

Public practice dataset from [Excelx.com](https://excelx.com/practice-data/sales-retail/). Contains 1,500 rows of retail sales transactions across five regions and multiple store locations, covering salesperson performance, discount and promotion impact, payment methods, and regional revenue breakdown.
